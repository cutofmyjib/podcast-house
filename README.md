Podcast House
=====================

Podcast House is podcast playlist project using React.
![](podcast-house-landingpage.jpg)

## Demo

```
git clone https://github.com/cutofmyjib/podcast-house.git
cd react-transform-boilerplate
npm install
npm start
open http://localhost:3000
```
